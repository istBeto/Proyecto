# COMANDOS PARA EJECUTAR EL PROYECTO BI

## 1. Abrir PowerShell en la carpeta ProyectoBI

Verificar Python:

python --version

---

## 2. Activar entorno virtual

.venv\Scripts\activate

Debe aparecer:

(.venv)

---

## 3. Ejecutar Dashboard

cd dashboard

python -m streamlit run app.py

o

streamlit run app.py

---

## 4. Abrir Dashboard

Si no abre automáticamente:

http://localhost:8501

---

## 5. Detener Dashboard

CTRL + C

---
